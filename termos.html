Termos

1. Objetivo da Plataforma

O Doglist é uma plataforma comunitária dedicada a:

• Ajudar a localizar animais perdidos
• Reportar animais encontrados
• Compartilhar localização de clínicas veterinárias
• Criar uma rede de apoio para animais perdidos

2. Uso Responsável

Ao utilizar o Doglist, você concorda em:

• Fornecer informações verdadeiras sobre animais perdidos/encontrados
• Não usar a plataforma para fins comerciais
• Respeitar a privacidade de outros usuários
• Manter as informações de localização das clínicas atualizadas

3. Privacidade e Dados

Coletamos apenas dados essenciais para:

• Identificação do usuário
• Localização aproximada para busca de animais
• Contato para comunicação entre usuários
• Mapeamento de clínicas veterinárias

4. Responsabilidades

O Doglist não se responsabiliza por:

• Veracidade das informações postadas por usuários
• Negociações ou acordos entre usuários
• Qualidade dos serviços veterinários indicados
• Custos relacionados a resgates ou tratamentos
