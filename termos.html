Termos e Condições

1. Objetivo da Plataforma

O Doglist é uma plataforma comunitária dedicada a:

  • Ajudar a localizar animais perdidos
  • Reportar animais encontrados
  • Partilhar localização de clínicas veterinárias
  • Criar uma rede de apoio para animais perdidos

2. Uso Responsável

Ao utilizar o Doglist, concordas em:

  • Fornecer informações verdadeiras sobre animais perdidos/encontrados
  • Não utilizar a plataforma para fins comerciais
  • Respeitar a privacidade de outros utilizadores
  • Manter as informações de localização das clínicas actualizadas

3. Privacidade e Dados

Recolhemos apenas dados essenciais para:

  • Identificação do utilizador
  • Localização aproximada para procura de animais
  • Contato para comunicação entre utilizadores
  • Mapeamento de clínicas veterinárias

4. Responsabilidades

O Doglist não se responsabiliza por:

  • Veracidade das informações publicadas por utilizadores
  • Negociações ou acordos entre utilizadores
  • Qualidade dos serviços veterinários indicados
  • Custos relacionados com resgates ou tratamentos
